# ML-Cardio-Disease-project

# Прогнозирование сердечно-сосудистых заболеваний

Индивидуальный проект по машинному обучению.

## Как запустить
1. Открыть в [Google Colab](ссылка на ваш Colab с share)
2. Run all

## Описание
- Датасет: Kaggle Cardiovascular Disease  
- Модели: Logistic Regression, Random Forest, XGBoost  
- Лучший ROC-AUC: 0.80
